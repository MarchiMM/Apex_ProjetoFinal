TABLES:

'*' -> Symbolizes foreign keys (FKs)

Company (Empresa)
    Name (Nome)

Person
    Type (Tipo: Pessoa Física ou Pessoa Jurídica)
        P : PhysicalPerson
        L : LegalPerson
    Name (Nome)
    Cpf
    Cnpj
    *FKCompany (FKEmpresa)
    Address (Endereço)
    Phone_Number (Número de telefone)
    E-mail

Employee (Funcionário)
    *FKPerson
    Role (Cargo)

Client (Cliente)
    *FKPerson
    Registration_Date (Data de cadastro)

Equipament (Equipamento)
    Type (Tipo)
    Brand (Marca)
    Model (Modelo)
    SerialNumber (Número de Série)

Taxation (Taxação)
    TaxDescription (Descrição do Imposto)
    Percentage (Porcentagem)
    EffectiveDate (Data de vigência)
    ExpirationDate (Data de expiração)

Request (Pedido)
    Status
        O : Open
        C : Closed
    *FKClient
    *FKEquipament
    *FKEmployee
    ServiceDescription (Descrição do serviço)
    *FKTaxation
